Reference list for Bayesian Data Analysis:

  • Statistical Rethinking from Richard McElreath: http://xcelab.net/rm/statistical-rethinking/
  • Bayesian Data Analysis from Andrew Gelman et al: http://www.stat.columbia.edu/~gelman/book/
  • Data Analysis using Regression and Multilevel/Hierarchical Models from Andrew Gelman and Jennifer Hill: http://www.stat.columbia.edu/~gelman/arm/
  • Bayesian Data Analysis in Ecology using Linear Models with R, BUGS and Stan from Franzi Korner-Nievergelt et al: https://www.elsevier.com/books/bayesian-data-analysis-in-ecology-using-linear-models-with-r-bugs-and-stan/korner-nievergelt/978-0-12-801370-0
  • Ecological Models and Data in R from Ben Bolker: https://ms.mcmaster.ca/~bolker/emdbook/index.html
